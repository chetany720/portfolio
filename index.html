Projects

Kidney MRI Image Segmentation Using AttentionVgg16Unet

This project aims to accurately segment kidney regions from MRI images using the AttentionVgg16Unet model. Combining the U-Net architecture with VGG16 as the backbone and an attention mechanism, the model improves feature extraction and focuses on relevant regions. The approach involves training on annotated MRI data, evaluating performance with metrics like Dice coefficient, and integrating the model into clinical tools for enhanced diagnostic support. The goal is to provide precise kidney segmentation to assist radiologists in medical analysis.

Kidney MRI Image Segmentation Using MultiResUnet

This project employs the MultiResUnet model to segment kidney regions from MRI images. By utilizing multi-resolution feature extraction, MultiResUnet improves the model's ability to capture both fine details and larger structures. The goal is to provide precise kidney segmentation to support accurate medical analysis and diagnosis.
